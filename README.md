# adminstates
Look up the States or Provinces or Admin Divisions or Regions of a country.

I found a free api site from https://github.com/kamikazechaser/administrative-divisions-db

Using this, I put together a fun code to look up all the States or Provinces of a country.

Need the 2 digit ISO code of the country to run this.

I added some in the countrymapping.json file.

You can update that file to add additional countries. Feel free to push it back into the git repo.
